# Emerald
Emerald is a simple script editor for `StateScripts` and `CustomBehaviors` in InfiltrationEngine.

## Usage
The plugin works immediately after installation with no setup required. Selecting an instance with an editable `CustomBehavior` or `StateScript` source automatically opens the script editor. For physical instances, a dropdown arrow is shown instead, allowing you to still make physical changes to the instance. 

Currently, Emerald includes the following highlighted features:
- Syntax Highlighting: Colors of tokens in the script will dynamically change based on their contents.
- Autocomplete Suggestions: As you type commands, a suggestions window will appear. You can navigate it using `↑` and `↓`. Pressing `Tab` or `Enter` accepts a suggestion, whilst `Escape` dismisses it.
- Customization: Albeit very bare-bones (I hate making settings), the plugin includes a settings widget allowing you to customize its appearance and behavior.

Emerald's Settings may be viewed by pressing its plugin button.
